
# WebSocket Real-Time Chat (WhatsApp Style)

Данный проект представляет собой веб-приложение для обмена сообщениями в реальном времени, разработанное в рамках курса «WebSockets Protocol Explained». Основной акцент сделан на сохранении пользовательских данных и стилизации под интерфейс популярного мессенджера.

## Основные возможности
* **Real-time Messaging:** Мгновенная передача сообщений без задержек и необходимости перезагрузки страницы.
* **User Persistence:** Ваше имя пользователя (User_xxxx) сохраняется в локальном хранилище браузера (localStorage). При обновлении страницы вы остаетесь под тем же аккаунтом.
* **Chat History:** Вся переписка сохраняется на сервере в файле messages.json. При входе в чат новые участники видят всю историю сообщений.
* **WhatsApp Style UI:** Дизайн с использованием характерной цветовой гаммы, «пузырей» сообщений и выравниванием (свои сообщения — справа, чужие — слева).

## Структура проекта
* `server.py` — WebSocket-сервер на Python.
* `index.html` — Клиентская часть (HTML + CSS + JavaScript в одном файле).
* `messages.json` — Файл базы данных для истории сообщений (создается автоматически).

## Инструкция по использованию

### 1. Установка зависимостей
Для работы сервера требуется библиотека `websockets`. Установите её с помощью пакетного менеджера pip:
```bash
pip install websockets
```

### 2. Запуск сервера

Перейдите в папку с проектом и запустите серверный скрипт:

```bash
python server.py

```

После запуска сервер будет ожидать подключений по адресу `ws://localhost:8080`. Не закрывайте терминал, пока пользуетесь чатом.

### 3. Запуск клиента

Для начала общения просто откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge).

* Чтобы протестировать чат, откройте этот же файл в нескольких разных вкладках.
* Попробуйте написать сообщение и убедитесь, что оно мгновенно появилось во всех окнах.
* Попробуйте перезагрузить страницу — благодаря localStorage и серверному JSON вы увидите, что ваш ник и история сообщений никуда не исчезли.

## Технический стек

* **Backend:** Python 3, WebSockets.
* **Frontend:** HTML5, CSS3 (Flexbox), JavaScript (Vanilla JS).
* **Storage:** JSON (Server-side), localStorage (Client-side).
